# Guía de uso

Este proyecto puede ejecutarse de dos formas: utilizando la extensión **Live Server** de Visual Studio Code o a través de un servidor local como **XAMPP**.

### Opción 1: Ejecutar con Live Server (Visual Studio Code)

1. Instala la extensión Live Server en Visual Studio Code.

2. Abre el proyecto en Visual Studio Code.

3. Navega y abre al archivo index.html.

4. Haz clic derecho en cualquier parte del código y selecciona Open with Live Server.

5. El proyecto se abrirá automáticamente en el navegador predeterminado.

### Opción 2: Ejecutar con XAMPP

1. Copia la carpeta del proyecto dentro del directorio htdocs de tu instalación de XAMPP.

2. Asegúrate de que el servidor Apache esté iniciado desde el panel de control de XAMPP.

3. Accede al proyecto desde el navegador usando la siguiente URL: http://localhost/[NOMBRE-DE-LA-CARPETA]
